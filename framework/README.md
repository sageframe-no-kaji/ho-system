# Framework

The core methodology of the Ho System.

## Contents

- `design-seed.md` — Founding document, philosophical grounding, theoretical foundations
- `structure/` — Structural elements (ho, project arc, tiered understanding, shu-ha-ri)
- `templates/` — Reusable templates for creating hos and devlogs

## Reading Order

1. Start with `design-seed.md` for the full philosophical and theoretical foundation
2. Review `structure/` to understand the key elements
3. Use `templates/` when creating your own hos
